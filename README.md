# EECONNECT Travel Data Portal

Lightweight static prototype for an outbound travel data package portal.

## Files

- `index.html` - page structure
- `styles.css` - visual styling and responsive layout
- `script.js` - country matching, zone logic, pricing toggles, and modal interactions

## Preview locally

Open `index.html` in a browser.
